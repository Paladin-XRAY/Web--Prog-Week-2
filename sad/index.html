anchor tag <a> The <a> tag defines a hyperlink, which is used to link from one page to another.

HTML Text Formatting Tags

Tags Symbol Use
Bold Tag <b> The HTML <b> element defines bold text, without any extra importance.
Strong Tag <strong> The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold.
italic Tag <i> The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic
emphasized Tag <em> The HTML <em> element defines emphasized text. The content inside is typically displayed in italic.
small Tag <small> The HTML <small> element defines smaller text
mark Tag <mark> The HTML <mark> element defines text that should be marked or highlighted
